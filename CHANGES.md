## [0.4.0] (2020-07-23)
* Added `root: true`
* Added `ignorePatterns` to lint dotfiles
* Required the latest ESLint version
* Extended the installation instructions

## [0.3.0] (2020-06-20)

* Updated the previous changelog entries

* Fixed the tag URL
